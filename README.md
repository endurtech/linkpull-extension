# LinkPull Extension
Grab any file straight from a URL. Highlight a link, right-click, or paste it in — LinkPull validates it and pulls it down. Like it? [Buy me a cup of coffee](https://endurtech.com/give-thanks/).

## Installation
Install this extension from the Chrome Web Store - to be submitted for publication.
